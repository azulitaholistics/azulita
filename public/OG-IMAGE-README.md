# Creating Your Open Graph Social Media Preview Image

## What You Need

Create an image file named `og-image.png` in this directory (`/public/`).

## Specifications

- **Dimensions**: 1200 x 630 pixels (required)
- **Format**: PNG or JPG
- **File size**: Under 8MB (under 5MB recommended)
- **File name**: `og-image.png`

## What to Include

Your preview image should contain:
1. **Logo**: The azulita bird logo (centered or top-left)
2. **Business Name**: "Azulita Holistics" in a large, readable font
3. **Tagline**: "Natural Healing & Wellness" or similar
4. **Brand Colors**: Use your teal/blue color (#5A9C8E) as the primary color
5. **Background**: Clean, simple background (solid color or subtle gradient)

## Design Tools

Create your image using:
- **Canva** (easiest): https://www.canva.com
  - Search for "Facebook Post" or "Twitter Post" templates
  - They're already sized at 1200x630
- **Figma** (free): https://www.figma.com
- **Adobe Photoshop** or similar

## Once Created

1. Save the file as `og-image.png` in `/public/`
2. No code changes needed - it's already configured!
3. Test it by sharing your site URL on Facebook or Twitter

## Testing Your Preview

- **Facebook Debugger**: https://developers.facebook.com/tools/debug/
- **Twitter Card Validator**: https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/

Just paste your URL and these tools will show you how your preview looks!
